# Transliteration Generator

Generate a clean transliteration matching the dataset's style.

## Rules
- Use consistent academic-style Latin transliteration.
- Keep long vowels and hamzah/apostrophe usage reasonable and consistent.
- Preserve phrasing and word order where practical.
- Do not over-explain.

## Output
Return a single `transliteration` string.
